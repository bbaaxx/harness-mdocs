mdocs command --help is a CLI-backed discovery surface for aggregate command payloads. It should remain plain text, exit 0, and show concrete --json examples for initiative.create, initiative.update, wiki.create, and wiki.update. initiative.update supports metadata changes under an updates object while progressNote remains top-level. wiki.update does not use an updates wrapper; changed fields such as content, lifecycle, confidence, and sourceInitiatives go at the top level with category and id.
